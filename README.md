# MachpIA - La IA de la Hassidout

Cette application est une intelligence artificielle experte en Hassidout (mouvement Loubavitch), conçue pour répondre avec sagesse et précision en utilisant les enseignements du Tanya et des Sichos du Rebbe.

## Versions disponibles

### 1. Version Web (Vite/React)
C'est la version complète de l'application avec toutes les fonctionnalités (génération d'images, thèmes avancés, etc.).
- **Installation** : `npm install`
- **Lancement** : `npm run dev`

### 2. Version Portable (Fichier HTML unique)
J'ai créé un fichier nommé `machpia.html` à la racine du projet.
- **Utilisation** : Ouvrez simplement le fichier `machpia.html` dans n'importe quel navigateur web.
- **Avantages** : Aucun serveur requis, fonctionne directement comme un fichier local.
- **Configuration** : Une clé API Gemini sera demandée à l'ouverture pour activer l'IA.

## Fonctionnalités
- Chat intelligent sur la Hassidout.
- Thèmes Clair/Sombre.
- Icône "Flamme" pour les réponses de l'IA.
- Interface responsive (Mobile/Desktop).
